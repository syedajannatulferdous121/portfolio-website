Dockerized Network Testbed Ongoing Research

Status: concept & design phase — no public repository or measured results yet.

A reproducible, container-based network testbed for protocol & service benchmarking. Planned: parameterized topologies via Docker Compose, link emulation with tc/netem (delay/jitter/loss/rate), automated load (iperf3 · wrk · k6), and a Prometheus + Grafana telemetry stack. One-command runs, CI-ready, seedable experiments.

Proposed stack
  • Docker & Compose
  • tc / netem · iproute2

  • Grafana Dashboards
  • iperf3 · wrk · k6
  • Make · Bash · Python
